# ACMS
## About the Project: 
Amazon Repairs

## Prerequisites:
Python 3 and Django 2.0.3 RestAPIFramework

## Getting Started:
1. Clone the directory on your local computer.
2. Extract the project.
3. cd ACMS-master
4. python3 manage.py migrate
5. python3 manage.py runserver
